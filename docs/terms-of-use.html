Description of Service

The Extension is a free tool that converts color palettes from coolors.co into Tailwind CSS utility class formats. The Extension operates locally in your browser and provides:

  • Automatic detection of Coolors palettes
  • Conversion of color values to Tailwind CSS format
  • Easy copying of converted color classes
  • Local storage of saved palettes
